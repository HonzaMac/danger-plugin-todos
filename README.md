> A danger-js plugin to list all todos/fixmes/etc added/changed in a PR

## Usage

Install:

```sh
yarn add danger-plugin-todos --dev
```

At a glance:

```js
// dangerfile.js
import todos from 'danger-plugin-todos'

todos()
```
